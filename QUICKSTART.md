# 🚀 빠른 시작 가이드

## 1분 안에 테스트하기!

### 1️⃣ 설치

```bash
cd test-extension
npm install
npm run build
```

### 2️⃣ 실행

**VS Code에서 `test-extension` 폴더를 열고 `F5` 누르기**

새 창(Extension Development Host)이 열립니다.

### 3️⃣ 테스트

새 창에서:

1. **`Ctrl+Shift+P`** 열기
2. **`Claude: Simulate`** 타이핑 후 엔터
3. 3-5번 반복
4. **왼쪽 하단 확인**: `⚠️ 입력 대기: 3` 표시됨!
5. **클릭** 또는 **`Ctrl+Shift+I`** 누르기
6. 🎉 첫 번째 터미널로 자동 이동!

## ⌨️ 핵심 단축키

- **`F5`**: Extension 실행 (디버그 모드)
- **`Ctrl+Shift+P`**: 명령 팔레트
- **`Ctrl+Shift+I`**: 다음 터미널로 이동 ⭐
